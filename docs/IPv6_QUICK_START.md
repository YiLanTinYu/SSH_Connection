# IPv6快速使用指南

## 快速开始

### 1. 启动应用程序

```bash
cd H3C_SSH
python3 main.py
```

### 2. 添加IPv6设备

#### 方法一：手动添加
1. 在左侧面板填写设备信息
2. IP地址输入IPv6地址，例如：`2001:db8::1`
3. 点击"添加设备"按钮

#### 方法二：Excel批量导入
1. 准备Excel文件，包含以下列：
   - 品牌 (brand)
   - IP地址 (ip)
   - 端口 (port)
   - 用户名 (username)
   - 密码 (password)
   - 设备名称 (name，可选)

2. 点击"导入Excel文件"按钮
3. 选择Excel文件

### 3. 连接设备

1. 添加设备后，点击"开始连接"按钮
2. 系统会自动连接所有设备（最多5个并发）
3. 实时查看连接进度和状态

### 4. 查看结果

- **设备列表**：显示所有设备的连接状态
- **连接日志**：显示详细的连接过程
- **统计信息**：连接完成后显示成功/失败统计

## IPv6地址格式

### 有效格式示例

```
# 完整格式
2001:0db8:0000:0000:0000:0000:0000:0001

# 压缩格式
2001:db8::1

# 链路本地地址
fe80::1

# 回环地址
::1

# 全局地址
2001:db8:85a3::8a2e:370:7334
```

### 特殊地址说明

| 地址类型 | 示例 | 说明 |
|---------|------|------|
| 回环地址 | ::1 | 本地测试使用 |
| 链路本地地址 | fe80::1 | 需要指定网络接口 |
| 私有地址 | fc00::1 | 内网使用 |
| 全局地址 | 2001:db8::1 | 公网使用 |

## 常见问题

### Q1: 如何验证IPv6地址格式？

系统会自动验证IPv6地址格式。如果输入无效地址，会显示错误提示。

### Q2: 链路本地地址如何使用？

链路本地地址（fe80::/10）需要指定网络接口，例如：
```
fe80::1%eth0
```

当前版本需要在地址中手动添加接口标识。

### Q3: IPv4和IPv6设备可以混合使用吗？

可以！系统完全支持IPv4和IPv6设备混合使用，可以同时连接不同IP版本的设备。

### Q4: 如何查看设备使用的是IPv4还是IPv6？

在设备列表中，"IP版本"列会显示每个设备使用的IP版本（IPv4或IPv6）。

### Q5: IPv6地址在日志中如何显示？

IPv6地址在日志中使用方括号显示，例如：`[2001:db8::1]`，便于区分IPv4地址。

## 故障排除

### 问题：IPv6连接失败

**可能原因**：
1. IPv6地址格式错误
2. 网络不支持IPv6
3. 防火墙阻止IPv6连接
4. 设备不支持IPv6

**解决方法**：
1. 验证IPv6地址格式
2. 检查网络IPv6支持
3. 检查防火墙规则
4. 确认设备IPv6支持

### 问题：地址验证失败

**可能原因**：
1. 地址格式不正确
2. 使用了不支持的地址类型

**解决方法**：
1. 检查地址格式
2. 使用有效的IPv6地址
3. 查看错误提示信息

### 问题：UI显示异常

**可能原因**：
1. 字体不支持IPv6字符
2. 显示格式问题

**解决方法**：
1. 使用系统默认字体
2. 检查UI配置
